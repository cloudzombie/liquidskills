What Agents Get Wrong

  • Chain ID is 999 (mainnet) and 998 (testnet) — not 1
  • RPC is https://rpc.hyperliquid.xyz/evm — not an Ethereum endpoint
  • Reads go to POST /info — writes to POST /exchange (signed)
  • Asset IDs: perps = index in meta.universe, spot = 10000 + index
  • Nonces are a rolling window — not linear. Use millisecond timestamps
  • Native token is HYPE (18 decimals) — not ETH, not WETH
  • Bridge address: 0x2222...2222 moves HYPE from HyperCore → HyperEVM
  • Priority fees are burned to zero address — not to validators
  • Cancun hardfork without blobs. EIP-1559 enabled. Base fees burned
