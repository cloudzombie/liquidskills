# LiquidSkills — OpenClaw Skill

This directory contains the OpenClaw-installable skill for LIQUIDSKILLS.

## Install

```bash
clawhub install liquidskills
```

Or point your agent directly at the skill file:

```
https://raw.githubusercontent.com/cloudzombie/liquidskills/main/openclaw-skill/SKILL.md
```

## What It Does

Gives your OpenClaw agent instant, accurate Hyperliquid development knowledge. Covers HyperEVM, HyperCore API, wallets, Solidity, DeFi, testing, security, and production deployment.

See [liquidskills.help](https://liquidskills.help) for the full skill directory.
